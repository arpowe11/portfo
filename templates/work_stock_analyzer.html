Historical Stock Analyzer

C#/.NET WinForms Application

This Stock Analyzer application allows a user to upload historical stock data in CSV format to view in a nice formatted way. The application displays the stocks in a candlestick form with both bullish and bearish trends. The user can also hover over any candlestick or volume line and see what the OHLC for each stick is. You can check out the source code for this project by going to the Stock Analyis repo located below.
